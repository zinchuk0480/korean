Категория желания

Первая основа глагола + 고 + 싶다
1. В повествовательных предложениях с подлежащим, выраженным 1-м лицом / в вопросительных предложениях с подлежащим, выраженным -м лицом (싶다)
2. В предложениях с подлежащим, выраженным местоимением 3-м лица (싶어하다) Пример:
저는 이 한복을 사고 싶습니다 (Я хочу купить этот ханбок)
한국에 가고 싶습니까? (Вы хотите поехать в Корею?)
그녀도 한복을 사고 싶어합니다 (Она тоже хочет купить ханбок)

Некоторые глаголы приобретают иное значение:
입다 > 입어 보다 (Надевать > Примерять)
멋다 > 먹어 보다 (Есть > Пробовать)

Категория попытки действия

Вторая основа глагола + 보다 Пример:
Конструкция: 쓰다 = 써 보다 / 읽다 = 읽어 보다
Настоящее время повелительное наклонение: 써 보세요. / 읽어 보세요. (попробуй написать / прочитать)
Прошедшее время: 써 봤어요. / 읽어 봤어요
(пробовал написать / прочитать) Будущее время время: 써 볼 거예요. / 읽어 볼 거예요 (попробую написать / прочитать)

Некоторые глаголы приобретают иное значение:
입다 > 입어 보다 (Надевать > Примерять)
멋다 > 먹어 보다 (Есть > Пробовать)

(이 / 가) 걸리다 Требуемое время в дороге

Пример:
삼십 분이 걸려요.
세 시간이 걸려요. С сущ. используется только именительный падеж

Суффикс вежливости —[으]시

На гласную (первая основа) + 시
На согласную (первая основа) + 으시
Глагол.
Примеры:
하다 = 하시다
찾다 = 찾으시다

Примеры со временем:
Настоящее время: 하세요. / 찾으세요.
Прошедшее время: 하셨어요. / 찾으셨어요.
Будущее время: 하실 거예요. / 찾으실 거예요.

Прилагательное.
Примеры:
예쁘다 = 예쁘시다
아름답다 = 아름우시다
Примеры со временем:
Настоящее время: 예쁘세요. / 아름우세요.
Прошедшее время: 예쁘셨어요. / 아름우셨어요.
Будущее время: 예쁘실 거예요. / 아름우실 거예요.

Определительная форма прилагательного

  1. Первая основа прилагательного на согласную + 은:
    맑다 > 맑 > 맑은 물. (Прозрачная вода)
  2. Первая основа прилагательного на гласную + ㄴ:
    세다 > 세 > 겐 바람 (Сильный ветер)
  3. Чередование ㄹ = 0 перед ㄴ, ㄹ, ㅂ, ㅅ - ㄹ исчезает при образовании грамматических форм:
    길다 > 길 > 긴 머리 (длинные волосы)
  4. Чередование ㅂ = 우 перед гласными конечной ㅂ изменяется на 우.
    По правилам определительной формы от основ, оканчивающихся на согл., используем окончание 은.
    춥다 > 춥 > 춥은, но по правилам чередования перед гласным ㅂ меняется на 우 = 춥 > 추우 > 추운
    변덕스럽다 > 변덕스러운
    무덥다 > 무더우 > 무더운
    덥다 > 더우 > 더운 (душный)
    아름답다 > 아름다우 > 아름다운 (жаркий)
  5. Чередование ㅎ = 0 빨갛다 > 빨간... сущ. (Красный)
    노랗다 > 노란... сущ. (Желтый)
    파랗다 > 파란... сущ. (Белый)
    하얗다 > 하얀... сущ. (Синий)
    까맣다 > 까만... сущ. (Черный)
    어떻다 > 어떤... сущ. (Какой?)

서 Предшествование

Описывание нескольких действий в одном предложении (поднявшись 서 идите прямо)

Пример:
올라가다 = 올라가서...
건너다 = 건너서... 올라가다 = 올라가서... 층우로 올라가서 왼쪽으로 가세요.

(이 / 가) 걸리다 Требуемое время в дороге

Пример:
삼십 분이 걸려요.

Какой (무슨 / 어떤 [어떻다] / 어느)

Какой? (В ответе - сущ.):
무슨 과일을 좋아해요? = 바나나를 좋아해요.
Какой? (В ответе - прилаг.):
어떤 과일을 좋아해요? = 단 과일을 좋아해요.
Какой? Который из?:
어느 과일을 좋아해요? = Указать какой из предоставленных на выбор.

을래요 / 갈래요 (Мнение говорящего / решение)

Когда хотим услышать мнение собеседника
Первая основа глагола = согл. + 을래요 / гласн + ㄹ래요 Пример:
먹다 = 먹을래요
가다 = 갈래요

아서 / 어서

Предложение №1. 그래서 Предложение №2.
Пример глагола + 서:
가다 = 가서서
배우다 = 배워서
Пример прилагательного + 서:
예쁘다 = 예뻐서
똑똑하다 = 똑똑해서
Пример существительного 이다 + 서:
согласная + 이어서 = 선생님이어서
гласная + 여서 = 의사여서

Отрицания 안 и 못

안 + глагол / прилаг.: 안 가요.
Пример: первая основа глагола + 지 않다
배우지 않아요.
못 + глагол: 못 가요.
Пример: первая основа глагола + 지 않다
배우지 못해요.

Союз 고 (И/А)

Для глагола и прилагательного используется в качестве союза «И», а для существительных как союз «А».
Первая основа глагол/прилаг. + 고 + глагол/прилаг. Пример глагол: 요리하고 공부해요.
Пример прилаг: 여자는 예쁘고 똑똑해요.
Пример сущ: 우리 할아버지는 선새ㅇ님이고 할머니는 의사입니다.

Союз 그런데(но / однако)

Предложение №1. 그런데 Предложение №2.
Пример: 오늘은 시간이 없어요. 그런데 내일 시간이 있을 거예요.

Союз 그리고 (И)

Предложение №1. 그리고 Предложение №2.
Пример: 어제 친구를 만났어요. 그리고 서점에 갔어요.
Пример: 저는 책을 샀어요. 그리고 저녁을 먹었어요.

Прошедшее время

았 / 었

Глаголы:
Вторая основа + ㅆ + 어요.
Пример: 먹다 (1 основа) => 먹어 (2 основа) => 먹었어요.

Прилагательные:
Вторая основа + ㅆ + 어요.
Пример: 예쁘다 (1 основа) => 예뻐 (2 основа) => 예뻤어요.

Существительные на Согласную:
Согласная + 이었 + 어요.
Пример: 선생님 => 선생님이었어요.

Существительные на Гласную:
Гласная + 였 + 어요.
Пример: 의사 => 의사였어요.

Повелительное наклонение

согласная + 으세요. / гласная + 세요.
Примеры:
먹다 - 먹으세요.
사다 - 사세요.
읽다 - 읽으세요.
가다 - 가세요.

Окончания

согласная + 이에요 / гласная + 예요 - неофициальная форма, утверждение или вопрос (зависит от интонации)
согласная + 습니까? / гласная + 입니까? - официальная вопросительная форма
согласная + 습니다 / гласная + 입니다 - официальная утвердительная форма

Чтение букв в подчиме

ㄳ, ㄵ, ㄼ, ㄽ, ㄾ, ㅄ - читается только первая согласная
ㄺ, ㄻ, ㄿ - читается только вторая
В обоих случаях читаются обе, если следующая буква ㅇ (вторая буква из подчима заменяет ее)

Правила чтения ㅅ и ㅢ

ㅅ и ㅆ - читаются как [щ и щщ] перед буквами ㅣ, ㅑ, ㅛ, ㅠ
ㅢ - читается как [ㅣ] после всех согласных, кроме ㅇ

Сонорные буквы

ㅇ, ㄴ, ㄹ, ㅁ - после этих букв происходит озвончение согласных
